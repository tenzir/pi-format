This release renames the npm package from pi-format to pi-formatter to resolve npm package-name conflicts and keep publishing reliable.

## 🐞 Bug fixes

### Package rename from pi-format to pi-formatter

`pi-format` has been renamed to `pi-formatter` for npm publishing because npm rejected `pi-format` as too similar to the existing package `d3-format`.
